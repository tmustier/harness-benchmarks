# GOV.UK style skill

This Agent Skill helps an agent write clear reports, guidance and documentation.

## Install

Copy the `govuk-style` directory into a supported Agent Skills location. You can also keep it in a repository's local skills directory.

## Use

Invoke `govuk-style` when a report or draft should be clear, front-loaded and accessible without losing technical detail.
